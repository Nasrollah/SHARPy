SHARPy

Simulation of High-Aspect Ratio Planes

Download this project as a .zip file Download this project as a tar.gz file

You are in the project pages for the framework for Simulation of High-Aspect-Ratio Planes. This is a simulation environment originally built at the Load Control & Aeroelastics Group at Imperial College London (http://www.imperial.ac.uk/aeroelastics).

The code is made available under a BSD license (https://github.com/rafapalacios/SHARPy/blob/master/license.txt). If you use it, we would ask you to cite the papers below to acknowledge our effort. The current distribution is in its beta version and there will be updates coming here regularly. If you wish to contribute or use the software, please do contact us at r.palacios@imperial.ac.uk.

References:

Hesse H., Palacios R. (2014) Reduced-Order Aeroelastic Models for the Dynamics of Maneuvering Flexible Aircraft, AIAA Journal (52):1717-1732 [http://dx.doi.org/10.2514/1.J052684]

Simpson R.J.S., Palacios R., Murua J. (2013) Induced-Drag Calculations in the Unsteady Vortex Lattice Method, AIAA Journal (51):1775-1779 [http://dx.doi.org/10.2514/1.J052136]

Murua J., Palacios R., Graham J.M.R. (2012) Applications of the unsteady vortex-lattice method in aircraft aeroelasticity and flight dynamics, Progress in Aerospace Sciences (55):46-72 [http://dx.doi.org/10.1016/j.paerosci.2012.06.001]
